the firmware.elf file is firmware used for Pixhawk V2.

upload this file to pixhawk.


to build this on your own environemnt do following 

cd ~/ardupilot/ArduCopter
make px4-v2-hexa

to upload this firmware do 
make mx4-v2-hexa-upload

get more information here : http://ardupilot.org/dev/docs/building-px4-for-linux-with-make.html
